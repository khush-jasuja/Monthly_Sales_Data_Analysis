# Sales Dashboard README

## Overview
This dashboard provides a complete sales performance analysis using Excel-based datasets. It helps visualize revenue trends, customer behavior, product performance, and profitability insights.

The dashboard is built using data from the `Sales_data_1.xlsx` workbook and includes multiple analytical sheets for reporting and visualization.

---

## Dataset Information
The workbook contains the following sheets:

| Sheet Name | Description |
|---|---|
| `Raw_data` | Main transactional dataset containing sales records |
| `Total_Revenue` | Overall business performance metrics |
| `Monthly_Revenue` | Revenue trend by month |
| `Product_Revenue` | Revenue breakdown by product category |
| `State_Revenue` | Revenue generated by state |
| `Gender_Revenue` | Revenue comparison by customer gender |
| `age_group_revenue` | Revenue distribution by customer age group |
| `Dashboard` | Main visualization and reporting dashboard |
